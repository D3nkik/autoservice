## Шаг 2 — Запустить установку

Подключитесь по SSH и выполните:
```bash
chmod +x ~/autoservice/setup.sh
bash ~/autoservice/setup.sh
```

Это автоматически:
1. Создаст таблицы в MySQL (с префиксом `asvc_`, не трогая твой Laravel)
2. Заполнит тестовыми данными (4 подъёмника, услуги, admin-аккаунт)
3. Соберёт backend (TypeScript → JavaScript)
4. Соберёт frontend (Next.js)
5. Запустит оба процесса через PM2

## Шаг 3 — Настроить nginx

Добавь в конфиг nginx для домена 24rasp.ru:
```nginx
# API backend
location /api/ {
    proxy_pass http://127.0.0.1:4000/api/;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-Proto $scheme;
}

# Frontend /auto
location /auto {
    proxy_pass http://127.0.0.1:3001;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-Proto $scheme;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
}

# Next.js статика
location /_next/ {
    proxy_pass http://127.0.0.1:3001/_next/;
    proxy_set_header Host $host;
}
```
